With this plugin you can change WooCommerce image placeholder with no code required!

== Description ==

This plugin allow you to change WooCommerce default image for product simply by choose from your media gallery.
Page option will appear in your WordPress backend menu, and here you can open the media gallery ad upload a new placeholder image.
You only have to choose an image from your library or your local folder and select the image.


PREMIUM FEATURES of NS Custom Placeholder Image for WooCommerce:

Choose a different placeholder per **categories**.
Choose a different placeholder per **tags**.

For more information about PREMIUM VERSION, visit official page <a href="http://www.nsthemes.com/product/custom-placeholder-image-for-woocommerce/" target="_blank">NsThemes Marketplace</a>.

== Installation ==

1. Upload the entire 'ns-custom-placeholder-image-for-woocommerce' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. View admin menu, then click on 'Cus Placeholder' and choose your image


== Frequently Asked Questions ==

= Code is required to use this plugin? =

No code require to use this plugin.

= Can I put an image from my local pc? =

Yes, you can use media images or upload your image.

= Can I reset default placeholder image? =

Yes, you can change set it by click on "Set to default" button.
